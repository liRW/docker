# 安装、卸载、更新、查看 、版本
```
#安装
pip install torch==2.1.2
#解压
pip uninstall torch==2.1.2
#更新到最新版本
pip install --upgrade torch 
# 查看已安装的包
pip list 
# 查看版本
pip --version
# 获取帮助文档
 pip  install --help
```
